# mithril-ts-webpack-boilerplate
My favorite mocking template

## Note
edit project's profiles in package.json.

## Structure
```
src/ (sources to edit)
  `-- index.ts
  `-- (any resources)
www/ (distributed public resources; auto generated)
  `-- index.html
  js/
    `-- app.js
```

## How to use
1. npm install
1. npm start
